islands 3.8.0

Стабильный срез библиотек islands-* под общим номером 3.8.0.

Общие изменения:

  • Исправлены XSS уязвимости в блоках input, select, b-form-attach
  • Блоки images, serp, serp-ru, video, b-country-flag, b-geolocation, b-head-logo, b-layout-table, b-mooa-panel, b-mooa, b-pager, b-related-list, b-static-text, i-ctr, i-error-reporter, i-geodata, i-https, i-oframebust, i-services-request-controller, l-head, l-page, notification-panel-notice, navigation объявлены устаревшими (deprecated) и будут удалены в islands@4.0.0
  • Блоки header2, tabs-menu, tabs-panes, tabs объявлены стабильными

islands-components 3.8.0

  • Устранена XSS уязвимость в блоке select
  • Разработана новая версия обще-портальной кнопки — button2, которая заменит собой button

  • В блоке input в поле placeholder появилась возможность использовать BEMJSON. Теперь можно передавать локализационные ключи

      {
    block: 'input',
    placeholder: {
    block: 'i-bem',
    elem: 'i18n',
    keyset: 'keyset',
    key: 'key'
    },
    ...
    }

  • В блоках alert, dropdown, select и suggest заменен block.destruct на BEM.DOM.destruct. Теперь вложенные блоки тоже уничтожаются.

  • В блок attach добавлены переводы на все поддерживаемые языки
  • В блоке checkbox элемент tick смещен на 4px вверх, чтобы компенсировать увеличение его высоты в версии 3.7.1. Теперь его не нужно дополнительно выравнивать.
  • Исправлена ошибка при уничтожении блока dropdown
  • Блоки tabs-menu, tabs-panes, tabs объявлены стабильными

islands-user 3.7.0

  • Дизайн блока ticker стал более плоским: пропала белая рамка вокруг индикатора.
    • У блока изменились размеры: он сдвинулся на 2px вверх и влево за счёт пустых областей
    • Высота и ширина уменьшились на 2px
  • В блок user, где используется ticker, были внесены изменения: индикатор сдвинулся вниз и вправо на 2px, что уравновесило изменения в блоке
  • Блок notification-panel-notice объявлен устаревшим и будет удален в следующей мажорной версии

islands-page 3.6.0

  • Устранена XSS уязвимость в блоке input
  • В блок search2 добавлен элемент hidden. Он представляет из себя не что иное как <input type='hidden'> и предназначен для добавления в форму скрытых полей, которые при отправке добавляются браузером в качестве cgi параметров.
  • Появилась возможность добавлять иконки в инпут (слева/справа от инпута), не мешая кнопке очистить ([X])
  • Блок header2 объявлен стабильным
  • Блок navigation объявлен устаревшим и будет удален в следующей мажорной версии

islands-icons 3.3.0

  • Блоки images, serp, serp-ru, video объявлены устаревшими и будут удалены в следующей мажорной версии
  • В блок service-icon добавлены иконки сервисов advertising, direct, metrika, site, webmaster

islands-romochka 3.6.1

  • Исправлена XSS уязвимость в блоке b-form-attach
  • Исправлены ошибки в блоках b-bage, b-slider-dots, i-counter, i-flashcookie, keycodes
  • Исправлены циклические mustDeps
  • В блоке i-services обновлены переводы
  • Блоки b-country-flag, b-geolocation, b-head-logo, b-layout-table, b-mooa-panel, b-mooa, b-pager, b-related-list, b-static-text, i-ctr, i-error-reporter, i-geodata, i-https, i-oframebust, i-services-request-controller, l-head, l-page объявлены устаревшими и будут удалены в следующей мажорной версии
